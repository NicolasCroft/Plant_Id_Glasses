How it works

01
Voice command
Say "Hey Siri, run Identify Plant." Siri triggers a custom Automator app that launches the script with full macOS camera and audio permissions.
02
Image capture
The attached Walfront Pro USB camera grabs a single frame using imagesnap, pointing wherever you're looking.
03
AI identification
The image is base64-encoded and sent to the Plant.id API, which returns the most likely species and a confidence percentage.
04
Audio feedback
The result is spoken through the OhO Sunshine glasses' built-in speakers via pyttsx3 TTS — e.g. "With 94.3% certainty, this is a Begonia rex."

Under the hood

The code

identify_once.py 
def identify_image(path):
    # Load + encode image as base64
    with Image.open(path) as img:
        buf = io.BytesIO()
        img.save(buf, format="JPEG")

    # Build Plant.id API request
    payload = {
        "images": [base64.b64encode(buf.getvalue()).decode()],
        "plant_details": ["common_names"],
        "modifiers": ["crops_fast", "similar_images"]
    }
    resp = requests.post(API_URL, headers=headers,
                         json=payload, timeout=20)

    # Extract top result
    top = resp.json()["suggestions"][0]
    return top["plant_details"]["common_names"][0], top["probability"]

def main():
    img_path = capture_image("HD Camera")
    name, prob = identify_image(img_path)
    phrase = f"With {prob*100:.1f}% certainty, this is a {name}."
    tts.say(phrase)
    tts.runAndWait()
Language

Python 3.11

Homebrew install, run via Automator wrapper for full macOS permissions and correct $PATH.

Vision API

Plant.id

REST API accepting base64 images, returning ranked species suggestions with confidence scores.

Audio

pyttsx3 TTS

Cross-platform text-to-speech. Female voice (Samantha), 120 WPM, piped through glasses speakers.

Capture

imagesnap

macOS CLI tool for single-frame camera capture. Targets the Walfront Pro by device name.

Trigger

Siri + Shortcuts

Shortcuts "Open App" action points to an Automator .app — avoids shell sandbox issues.

Image

Pillow

In-memory JPEG re-encoding before base64 serialization, keeping the API payload compact.

What's next

Beyond plant ID

Plant identification is just the proof of concept. By swapping the Plant.id API for any vision or AI service, the same glasses platform can tackle far bigger problems.

🩺

Skin condition screening

Reliable visual screening for common dermatological conditions — no more googling symptoms and getting worst-case results.

🧠

Name recall assist

Facial recognition to quietly whisper someone's name — potentially transformative for people with memory conditions like early Alzheimer's.

👁️

Assistive vision

Help the partially sighted understand their surroundings — describe objects, read text, identify hazards, all through audio.

🤖

General AI assistant

A wearable window to any AI model. Look at anything — a contract, a machine, a dish — and ask a question about it.
